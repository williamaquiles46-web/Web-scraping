# 🔍 Comparador de Preços: Mercado Livre & Amazon

Este é um projeto de **Web Scraping** desenvolvido em Python para automatizar a busca e comparação de preços entre o Mercado Livre e a Amazon. O objetivo é ajudar a encontrar a melhor oferta para produtos como o **iPhone 16**.

## 🚀 Tecnologias Utilizadas

* **Python 3.10+**
* **Selenium**: Para automação de navegador e bypass de carregamento dinâmico.
* **BeautifulSoup4**: Para extração e parsing dos dados HTML.
* **Pandas**: Para tratamento de dados e organização.
* **Openpyxl**: Para exportação dos relatórios em Excel.
* **WebDriver Manager**: Para gerenciamento automático do ChromeDriver.
